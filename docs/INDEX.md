## üì¶ Package Formats

SnapWiz supports **4 package formats:**
- `.deb` - Debian/Ubuntu packages
- `.rpm` - Fedora/RHEL packages  
- `.snap` - Universal Snap packages
- `.flatpak` - Universal Flatpak bundles

See [FEATURES_AND_IMPLEMENTATION.md](FEATURES_AND_IMPLEMENTATION.md#supported-package-formats) for details.
